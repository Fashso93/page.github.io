Documental Código Linux


El documental comienza con una gran declaración de Richard Stallman, comparando el desarrollo de software con recetas. Es muy claro introducir al espectador en la filosofía del software libre. A partir de ahí, comienza la historia.


En 1983, Stallman inició el proyecto GNU para crear un sistema operativo compatible con los estándares UNIX y POSIX. Con este fin, fundó la Free Software Foundation en 1985 y obtuvo la licencia GNU GPL. Con este marco legal, el software gratuito ha comenzado a desarrollar enormes etapas. De esa manera, en 1990, hubo un código escrito que casi se podría construir porque dijo el sistema operativo.
